# java.time이란?

JDK 8부터 사용할 수 있는 날짜·시간 처리 패키지이다.

기존의 `Date`, `Calendar`보다 날짜와 시간을 명확하게 구분하고<br>
안전하게 처리할 수 있도록 만들어졌다.

`LocalDate`, `LocalTime`, `LocalDateTime`은 시간대 정보가 없으며,<br>
서버와 DB가 **같은 지역 시간을 기준**으로 동작하는 일반적인 프로젝트에서 사용할 수 있다.

### 종류

```java
LocalDate      // 날짜만: 2026-06-19
LocalTime      // 시간만: 14:35:20
LocalDateTime  // 날짜 + 시간: 2026-06-19T14:35:20
```

### 기존 타입들과의 차이점

| 타입               | 주요 의미                      | 예시                    |
| ---------------- | -------------------------- | --------------------- |
| `java.sql.Date`  | JDBC에서 날짜만 표현하기 위한 타입      | `2026-06-19`          |
| `java.util.Date` | 특정 시점을 나타내는 구형 날짜·시간 타입    | `2026-06-19 14:30:20` |
| `LocalDateTime`  | 시간대 없는 날짜와 시간을 나타내는 최신 API | `2026-06-19T14:30:20` |

<br>

---

### 프로젝트 적용

Oracle의 `DATE`는 날짜뿐 아니라 시·분·초까지 저장하므로, <br>
`CREATED_AT`, `UPDATED_AT`은 `LocalDateTime`으로 매핑한다.
